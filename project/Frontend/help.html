📘 EduBot

Help & Guide

Learn how to use Chat, Programs, Quiz, Flashcards and Progress features effectively 🚀

💬 Chat Module

  • Go to the Chat page from sidebar.
  • Type your programming or theory question.
  • Click Send or press Enter.
  • Bot replies instantly below your message.
  • Your chat history loads automatically.

💻 Programs Module

  • Open the Program page.
  • Select a programming topic.
  • View code examples with explanations.
  • Practice the code in your local editor.

📝 Quiz Module

  • Navigate to the Quiz page.
  • Select a topic to attempt questions.
  • Choose correct answers carefully.
  • Submit quiz to see your score instantly.
  • Scores are stored for progress tracking.

🧠 Flashcards Module

  • Go to Flashcards section.
  • Click the card to flip and see answer.
  • Use Next and Previous buttons to navigate.
  • Best for quick revision before exams.

📊 Progress Module

  • Open Progress page.
  • View quiz scores and performance stats.
  • Track your improvement over time.
